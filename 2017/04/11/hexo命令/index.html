hexo命令

Hexo
  • Hexo

一些hexo常用的命令


hexo n

1、 hexo new  ”pageName”

  • 用于新建一篇文章，会直接呈现到主页面
  • 文件在 主文件/source/_posts 下

hexo n page

2、 hexo new page   “pageName”

  • 用于新建一个页面，如“关于我”
  • 执行命令后会在 主文件/source/ 下新建这个页面

hexo g

3、 hexo generate

  • 生成静态页面至public目录
  • 部署之前的操作

hexo s

4、 hexo server

  • 将页面部署到本地的端口上，默认是 http://localhost:4000/
  • 主要用于部署之前的调试，确认后再部署
  • 调试时，该命令只需要执行一次，以后的改动只需刷新页面即可
  • 退出调试，ctrl+c

hexo d

5、 hexo deploy

  • 部署页面

hexo clean

6、 hexo clean

  • 清理缓存，效果类似页面强制刷新
  • 如果部署后的页面和本地调试不一致，你就需要使用这个命令来清理下再部署

部署操作

1、hexo clean
2、hexo generate
3、hexo deploy
简单方法 hexo g -d
